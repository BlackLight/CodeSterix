Play <something>

A way to interact with the Spotify Web API through voice

How to

  1. Enable the microphone (you will see a notification on the top of this page)
  2. Say one of these commands:
    • Play <a song name> (e.g. Play "Wasted")
    • Play <a song name> by <an artist name> (e.g. Play "Barbie Girl" by "Aqua")

More info

Check out the source code for more information. This demo uses annyang and the Spotify Web API.
